# コミュニティサービス構想

## サービス概要
オープンな話題を気軽に話せるコミュニティサービス。
ユーザーが本来の自分でいられる居場所を作ることを目的とする。

## コンセプト
自分の家や部屋を持てるような、少し遊び心のあるコミュニティ。
昔の夢里村のように、ただ話すだけでなく「自分の居場所を育てる」楽しさを入れる。

## 主な機能案
- ユーザー登録
- プロフィール作成
- 掲示板・トピック投稿
- コメント機能
- 自分の部屋・家ページ
- いいね・リアクション
- 通報・ブロック
- 管理者による投稿管理
